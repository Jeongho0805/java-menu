## 기능 목록

### 입출력 관련 기능
  * [ ] 코치 이름 입력
  * [ ] 각 코치별 못먹는 메뉴 입력
  * [ ] 메뉴 추천 결과 출력

<br>

### 비즈니스 로직 관련 기능
  * [ ] 카테코리별 메뉴 데이터 세팅
  * [ ] 코치 이름 입력에 따른 코치 객체 저장
  * [ ] 각 코치별 못먹는 메뉴 저장
  * [ ]

<br>

### 예외 처리 관련 
  * [ ] 코치 이름이 2~4글자가 아닐 경우
  * [ ] 코치 인원이 2~5명이 아닐 경우
  * [ ] 코치가 먹지 못하는 메뉴가 0~2개가 아닌경우

<br>

### 주의 사항!!! 
- 한주에 같은 카테고리는 2개까지 
- 각 코치에게 한주에 중복되지 않는 메뉴 추천
- 메뉴는 최초 제공한 목록 그대로를 전달!
